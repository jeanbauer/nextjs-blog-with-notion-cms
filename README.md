## Começando

Primeiro, instale as dependências:

```bash
npm install
# ou
yarn
```

Depois, inicie a aplicação:

```bash
npm run dev
# ou
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu browser para ver o resultado.
